File Format (.plux)
===================
Bars seperated by |

Notes seperated by :

Data is 
1-5 select string
A-Z set fretting and pluck that string this note
+   add hammer on 1 (multiple)
-   sub pull off 1 (multiple)
>   slide 1 (multiple)
/ 	go to next note.

Assignments are lines containing

def := constant

Simples .banjo (format = 1)
===========================

<chord> := 01223 is fretting for string 1-string 5
<bars> are | seperated

1-5 means strum 1-5 P plucks 1+5 [<chord>] switches chords. Defaults to G and keeps going unless otherwise
All 8 beat bars
