Erbsen Format
=============

xxxn - & 		format. 2 beats to a bar. (C)hord insertions. 

Options:

(d)rone : 		add drone notes on string 1 to any half-beat
(p)inch : 		add a pinch after any single beat
(c)hord : 		replace any chords with the actual notes.
(a)lternate : 	play the alternate thumb roll (x251)
(f)orward : 	play the forward roll x15x15x1 (with the forward fudge)
(p)egleg : 		play x_51
(r)everse : 	play x21512x1
(b)reakdown : 	play x1x15x15
(m)elody : 		insert the melody.

(rolls only on single notes)

done in bar units.

@<name> = <letters> , <x>.erbsen is generated as <x>_<name>.plux
multiples specify various types

default : beats (2), name, notes (8)
